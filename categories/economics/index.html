心路介绍

其实在本科学初级微观宏观的时候，我觉得经济学挺无聊的: 就那么几个数学公式还倒腾半天, 你把所有的基础公式摆给我, 我很快能把结论全部给你推导出来。这也导致了我一开始对经济学并不上心, 毕竟只要数理能力够了, 经济学的考试就是很简单的事情。

说来你们可能不信，我对经济学产生兴趣正是我在二年级学马克思主义基本原理 (后半段) 的时候。在那门课上，真正派上用场的是推理与逻辑。我惊讶于一层层严密的推理是如何一步步得到一个个社会问题的解，而在这之前我很少接受这样的逻辑教育。于是我开始思考，或许社会问题的背后是逻辑而非数学符号，数学只是一种语言载体而已。

经济学也是，表面的数学符号并没有什么意义，真正有价值的一直都是背后的故事。我想这对于任何一位热爱理论推演的人都是无法拒绝的诱惑。就这样，我开始思考偏实务化的金融学可能不是我应该追求的 (我要指出的是，即便金融学是很数理化的，相比之下它依旧很实务。对我而言，纯推数学公式和纯推理具有本质区别)。而这也就成为了我日后转专业的动机来源。后来我自学了中级微观 (Walter Nicholson) 与中级宏观 (David Romer)，为我读经济学做了进一步的准备。

给后来人的话: 如果你想要学好经济学

从我个人的经验来看, 可能你大致需要这些理论知识。这些知识在我看来是做学问的必备(或许该说预备)知识。颜色深浅对应初学的难度高低, 越浅越易；如果涉及箭头，则箭头右边的颜色代表在掌握了箭头左边知识后学习的难度:

数学

基础级别: 微积分, 线性代数, 数理统计, 集合论, (微积分 -> 常微分的计算)
进阶级别: (集合论 + 微积分 -> 测度论), (测度论 + 数理统计 ->概率论), (集合论 -> 实分析)
核心级别: (实分析 + 线性代数 -> 最值问题), (实分析 + 测度论 -> 泛函分析方程), (泛函分析方程 -> 动态最优规划)

经济学

初级级别: 经济学原理(曼昆), 宏观经济学(曼昆), 计量经济学导论(伍德里奇)
进阶级别: 经济学原理(曼昆) -> 微观经济学现代观点(范里安) 或 微观经济理论(尼科尔森), 宏观经济学(曼昆) -> 高级宏观经济学(罗默), 计量经济学导论(伍德里奇) -> Probability and Statistics for Economists (Hansen)
核心级别: 微观经济学现代观点(范里安) 或 微观经济理论(尼科尔森) -> 微观经济理论(MWG) -> A Course in Game Theory (O&R), 高级宏观经济学(罗默) -> Introduction to Modern Economic Growth (Acemoglu) -> Recursive Macroeconomic Theory (Ljungqvist)

叠甲: 仅代表我个人的意见, 不代表对所有学习者都生效
